# Archive

Deprecated, superseded, or retired STUDIO work — old prompt drafts, abandoned project folders, and experiments that should not be used in active production.

**Move here when:**
- A prompt or template is replaced by a newer version in `Prompts/` or `Templates/`
- A project is cancelled or merged into another production
- Research is outdated but worth keeping for history

**Do not** reference archive paths in active workflows. Promote stable material back to `Prompts/`, `Canons/`, or `Research/` only after review.
