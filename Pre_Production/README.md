# Pre-Production

Script → Breakdown → Storyboard → Shot List → Schedule → Budget

## Subfolders
- `Script/` — Screenplay, treatments, revisions (Director authority)
- `Breakdowns/` — Scene/element breakdowns
- `Storyboards/` — Visual planning
- `Shot_Lists/` — Coverage plans
- `Scheduling/` — Stripboard, day-out-of-days
- `Budget/` — Top-sheet, line items

## Gate
Legal Gate + Slate approval before locking script for production.
