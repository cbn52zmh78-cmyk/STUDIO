# Prompt Library

Reusable, production-ready prompts — scene templates, shot prompts, character blocks, and technique-specific libraries copied into active projects.

**Use when:** You have a tested prompt ready to reuse across productions.

**Not for:** Layer-wide system instructions (see `../system/`) or raw research notes (see `../../Research/`).

Copy from here into `Projects/<production>/` or reference by path in director sessions.
