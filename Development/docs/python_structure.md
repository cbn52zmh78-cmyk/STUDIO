# Python Structure in STUDIO Repo

This repository focuses on cinematic techniques, prompting research, and visual references.

## Purpose

The Python layer in STUDIO is intentionally lightweight. It helps organize:

- Cinematic techniques by director/style
- Prompt templates and libraries
- Visual reference management

It is designed to support consistency and research organization rather than heavy agent runtime (that belongs in the AI repo).

## Structure

- `studio/techniques/` — Specific director and style modules (Wes Anderson, Sean Baker, Giallo, Sofia Coppola, etc.)
- `studio/prompting/` — Centralized prompt templates and helpers
- `studio/references/` — Simple reference management utilities

## Getting Started

```bash
pip install -e .
```
