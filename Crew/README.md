# Crew Departments

Each subfolder holds crew lists, rate cards, contact sheets, and department-specific notes for a production.

| Subfolder | Department |
|-----------|------------|
| `Camera/` | DP, operators, AC, DIT |
| `Lighting/` | Gaffer, best boy, electricians |
| `Sound/` | Production sound mixer, boom |
| `Art/` | Production designer liaison |
| `Costume/` | Wardrobe supervisor |
| `HMU/` | Hair, makeup, prosthetics |
| `Production_Management/` | AD team, coordinators |
| `Transport/` | Drivers, picture vehicles |

Producer signs off crew before first day of principal photography.
