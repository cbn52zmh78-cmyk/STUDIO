# Productions

Active STUDIO deliverables — one folder per show, campaign, or scene chain.

## Zones

| Folder | Use |
|--------|-----|
| **Narrative/** | Long-form fiction — PI_Story, Plantagenet test scenes, SPIT |
| **History/** | Documentary figures ingested from `History/figures/*/research/studio_handoff.md` |
| **GFE/** | Per-actress scene chains beyond casting (`Cast/GFE/` holds assets) |
| **Editorial/** | Magazine campaigns beyond model folders |
| **_Scene_Production_Kit/** | **Clone this** for every new scene |

## New scene workflow

1. Copy `_Scene_Production_Kit/` → `{Zone}/{project_name}/`
2. Fill `QC_notes.md` and add plates to `plates/`
3. Generate packs into `Pipeline/`
4. Render → `renders/approved/` with sidecar
5. Log compliance in `Producers_Office/Compliance_Reports/`
