# History Productions

Ingest handoffs from `History/figures/{slug}/research/studio_handoff.md`.

Each figure gets a subfolder here with:

- `core_prompt.txt` — from handoff
- `render_plan.md` — shot sequence
- `plates/` — reference stills
- `packs/` — Pipeline exports linked by path
