# Post-Production

Edit → Color → Sound → VFX → Deliverables

## Subfolders
- `Edit/` — Assembly, director's cut, producer's cut
- `Color/` — LUTs, grades, dailies color
- `Sound/` — Mix, design, M&E
- `VFX/` — Compositing, cleanup
- `Deliverables/` — Masters per `Distribution/` specs

## Renders policy
Approved masters live in `renders/approved/` only after Legal GREEN and talent `agency_ready`.
