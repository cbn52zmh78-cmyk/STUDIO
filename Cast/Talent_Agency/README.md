# Talent Agency

Performance study layer for all Cast talent — actors roster, GFE, MAGAZINE editorial models.

| Folder | Purpose |
|--------|---------|
| **Performance_Studies/** | Per-talent `performance_study.json` + scene logs |
| **Scoring_Rubrics/** | Locked rubric definitions |
| **Reports/** | Agency meeting reports |
| **Pairing_Notes/** | Chemistry / casting pairing research |
| **roster_index.json** | Auto-generated roster manifest |

**Charter:** [`AGENCY_CHARTER.md`](AGENCY_CHARTER.md)

**Tools:** `artifacts/talent/performance_study_manager.py`
