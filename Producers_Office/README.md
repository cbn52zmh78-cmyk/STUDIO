| Folder | Purpose |
|--------|---------|

## Daily Producer checklist

1. **Gate 0 first** — Legal Gate on every scene, video, brief, client ask (AI law + mass dissemination + CARA)
2. Slate status current
3. Talent `agency_ready` before hero generation
4. Call sheet open for every Director session
5. **RED = stop.** Tell Director and client. No exceptions.
