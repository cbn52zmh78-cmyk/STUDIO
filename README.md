# STUDIO — Cinematic Techniques, Visual Storytelling & Prompting Research

**NEXUS-coordinated repo** for cinematic production systems.

## Current State
Mature infrastructure for cinematic techniques, prompting research, and direct consumption of Visualization Federation animation packages.

### Key Capabilities
- Cinematic style systems (impasto, free-cam, negative space, dollhouse, one-take, J/L-cuts)
- Prompt engineering & canon management
- AI Federation animation handoff consumption
- Physics-accurate motion & F1-style broadcast aesthetics
- Production workflows, templates, and review tools

## Quick Start
```bash
pip install -e .
python examples/consume_ai_handoff.py
```

See `docs/getting_started.md` and NEXUS registry for ecosystem coordination.
